# Dataset Generator

This folder contains Python scripts used to generate the complete retail dataset for the 60 Days PySpark & Spark SQL Challenge.

## Generated Tables

- customers.csv
- products.csv
- stores.csv
- orders.csv
- order_items.csv
- payments.csv
- inventory.csv

## Output Location

data/raw/

## Features

- Realistic business data
- Foreign key relationships
- Intentional dirty data
- Random but reproducible records
- Easy to scale from thousands to millions of rows
